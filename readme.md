## 路由分配

| 请求方法 | 请求路径      | get参数 | post参数 | 用途     |
| -------- | ------------- | ------- | -------- | -------- |
| GET      | /student      | -       | -        | 首页     |
| GET      | /student/add  | -       | -        | 添加页面 |
| POST     | /student/add  | -       | stu对象  | 添加用户 |
| GET      | /student/edit | id      | -        | 编辑页面 |
| POST     | /student/edit | -       | stu对象  | 编辑提交 |
| GET      | /student/del  | id      | -        | 删除用户 |
